nvme-wdc-drive-log(1)
=====================

NAME
----
nvme-wdc-drive-log - Retrieve hgst device's drive log and save to file.

SYNOPSIS
--------
[verse]
'nvme wdc drive-log' <device> [--output-file=<FILE>, -o <FILE>]

DESCRIPTION
-----------
For the NVMe device given, sends the wdc vendor unique drive log
request and saves the result to a file.

The <device> parameter is mandatory and may be either the NVMe
character device (ex: /dev/nvme0), or a namespace block device (ex:
/dev/nvme0n1).

This will only work on wdc devices. Results for any other device are
undefined.

OPTIONS
-------
-o <FILE>::
--output-file=<FILE>::
	Output file; defaults to device serial number


EXAMPLES
--------
* Gets the drive log from the device and saves to default file :
+
------------
# nvme wdc drive-log /dev/nvme0
------------
* Gets the drive log from the device and saves to defined file :
+
------------
# nvme wdc drive-log /dev/nvme0 -o wdc_drive_log
------------
